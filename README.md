# News-Sentiment-Analysis
Automated financial sentiment analysis using DistilRoBERTa and BeautifulSoup, designed for future price-regression testing.
​This project provides a robust framework for quantifying the sentiment of financial news headlines in real-time. By automating the data retrieval and classification process, the engine converts qualitative text into structured quantitative metrics. The core of the system is a Python-based pipeline that scrapes live data and processes it through specialized machine learning models to identify market mood.
​The technical implementation utilizes the BeautifulSoup4 library for web scraping and the transformers library to access the pre-trained distilroberta-finetuned-financial-news-sentiment-analysis model. To ensure operational stability, the script includes custom browser headers and randomized request intervals to manage rate-limiting. All processed data is exported into a structured JSON format, including ticker-specific sentiment labels and confidence scores.
​Future development will focus on the statistical analysis of this data. The next phase involves merging these sentiment scores with historical stock price time-series to perform regression analysis. This will allow for the testing of lead-lag correlations and behavioral economic hypotheses regarding the impact of news sentiment on asset price volatility.
